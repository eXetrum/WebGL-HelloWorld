Description: This program displays a cool morph of the 2D Sierpinksi gasket !
Functionality:
  • Each vertex of the original 2D gasket is assigned a color based on its (x,y) location.
    Rule for mapping colors:
    • red: (1 - Math.sin(x)) / norm
    • green: g = (1 + Math.cos(y) * x) / norm
    • blue: b = (1 - Math.cos(y)) / norm
      • where norm = sqrt(x^2 + y^2)
  • Each vertex is rotated based on its position to origin using conversion to polar form
    x' = x * cos(φ)-y*sin(φ)+x0
    y' = x * sin(φ)+y*cos(φ)+x0
    where (x, y) - current vertex position,
    φ - angle of point (actualy vector angle based on vertex coords)
    and (x', y') - new position

 Params:
  • NumTimesToSubdivide - how much times divide triangle
  • Morphing controlled by θ angle using slider to interactive changing
